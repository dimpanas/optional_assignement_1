# optional_assignement_1

# Προσομοίωση Ρυθμών TE σε Ορθογωνικό Κυματοδηγό

Το πρόγραμμα αυτό υλοποιεί τον υπολογισμό και την τρισδιάστατη (3D) απεικόνιση των ηλεκτρομαγνητικών πεδίων και της διαδιδόμενης ισχύος σε ορθογωνικούς κυματοδηγούς, σύμφωνα με τις αρχές της Μικροκυματικής Θεωρίας.

## Προαπαιτούμενα

Για τη διαχείριση των βιβλιοθηκών και την εκτέλεση του κώδικα, απαιτείται η εγκατάσταση του **uv** (ένας σύγχρονος και ταχύτατος package manager για Python).

### Εγκατάσταση του uv
- **macOS/Linux:**

```bash  
  curl -LsSf [https://astral.sh/uv/install.sh](https://astral.sh/uv/install.sh) | sh
  ```


- Windows (PowerShell):

PowerShell
```bash
powershell -ExecutionPolicy ByPass -c "irm [https://astral.sh/uv/install.ps1](https://astral.sh/uv/install.ps1) | iex"
```


Οδηγίες Εγκατάστασης και Εκτέλεσης
Η εκτέλεση μέσω uv είναι η πιο απλή διαδικασία, καθώς το εργαλείο αναλαμβάνει αυτόματα τη δημιουργία προσωρινού περιβάλλοντος και την εγκατάσταση των βιβλιοθηκών.

Κατεβάστε τον κώδικα:


```bash
git clone [https://github.com/dimpanas/optional_assignement_1.git](https://github.com/dimpanas/optional_assignement_1.git)
cd optional_assignement_1 
```


Εκτελέστε το πρόγραμμα: Τρέξτε την παρακάτω εντολή στο τερματικό σας:

```bash
uv run --with numpy --with matplotlib main.py
```





Τι περιλαμβάνει η προσομοίωση
Με την εκτέλεση του κώδικα, θα παραχθούν αυτόματα τα εξής διαγράμματα:

1) 3D Απεικόνιση Διατομής (z=0): Το μέτρο του ηλεκτρικού πεδίου ∣E∣ για τον ρυθμό TE10 στη συχνότητα των 7 GHz.
2) 3D Διάδοση κατά μήκος του z: Η διαδιδόμενη ισχύς P για τον ρυθμό TE10 στις συχνότητες 2, 5, 6, 7, 10, 15 και 20 GHz.
3) Ανάλυση Ρυθμών στα 15 GHz: Σύγκριση της κατανομής ισχύος για τους ρυθμούς TE10, TE20, TE01 και TE11
​



Σημειώσεις Φυσικής -> 

- Συχνότητα Αποκοπής (fc): Για τις διαστάσεις του κυματοδηγού (a=2.286 cm), η αποκοπή του TE10 είναι στα 6.56 GHz.

- Αποσβενύμενα Κύματα: Στις συχνότητες 2, 5 και 6 GHz η ισχύς είναι μηδενική, καθώς το κύμα βρίσκεται σε κατάσταση αποκοπής (cutoff).

- Διαδιδόμενα Κύματα: Για f>6.56 GHz, το κύμα μεταφέρει ενέργεια κανονικά κατά μήκος του άξονα z.
